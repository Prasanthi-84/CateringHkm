Payment Flow (UPI/Link/Cash)

Collect Now:

If % selected, compute collectAmount = round(total * percent/100).
If custom selected, use provided collectAmount.


UPI Manual: require UTR field → mark Succeeded instantly; store utr.
Gateway Link: create UPI‑only link (via PaymentProvider stub), send WhatsApp with link, store as Initiated. On webhook, update Payment to Succeeded/Failed and recompute balance.
Cash: require collectorName & collectorPhone.

Invoices should reflect partials: total, amountPaid, balanceDue.
